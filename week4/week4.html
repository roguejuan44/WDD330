A form allows the user to input information, which in the past, was immediately sent to a server and processed using PHP or Ruby. However, modern convention is to process the information on the front end with JS before sending it to the server.

Select

The select element creates a drop down list the user can select from. If you add the multiple attribute to the HTML, it will allow the user to select multiple items from this drop down list. The selected attribute sets the default selection. The list is built like so.


Textarea

Textarea is an element that allows the user to input much longer text strings than the normal input text element. This is great for maybe a contact us or a technical support page where you're sending an email or submitting a question. You can set the rows and cols in the HTML, or set the width and height in CSS.
