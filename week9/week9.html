Week 9

The Browser Object Model

Try not to use window objects

Dialogs

  • window.alert()
  • window.confirm()
  • window.prompt() - not considered user friendly as they are intrusive and stop everything else that is running
  • window.navigator.userAgent - returns browser/OS information
  • window.location.href - url string
  • window.history.go(x) - go back (-1), forwards (1), or reload (0).

Create a popup

const popup = window.open('https://sitepoint.com','SitePoint','width=400,height=400,resizable=yes');

popup.close()

COOKIES

document.cookie = 'info'

HTML 5 API

Use data- attribute to create custom attributes.

Local Storage: localStorage.setItem('name', 'Walter White');

Retrieve stored data: localStorage.getItem('name');

Delete: delete localStorage.name;

Clear: localStorage.clear();

Multimedia

  • video.play()
  • video.pause()
  • video.muted
  • video.volume
  • video.currentTime
  • video.playbackRate
  • video.loop
  • video.duration
